Welcome to Community Watch

A comprehensive crime reporting and management system for our community.

System Features

Citizen Portal

  • Report crimes and incidents
  • Track report status
  • View incident hotspot map
  • Manage personal profile

Authority Portal (Ajk Kampung)

  • Manage incident reports
  • Take action on reports
  • Update report status
  • View analytics dashboard

Admin Portal

  • Manage authority accounts
  • Monitor all activities
  • System analytics

Database & Setup

Installation Steps

  1. Clone repository ke komputer anda.
  2. Pastikan PostgreSQL telah dipasang.
  3. Jalankan perintah: psql -d db_name -f schema.sql
  4. Mulakan server dengan: npm start
